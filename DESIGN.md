# OfflineChat - 离线聊天应用设计文档

## 气质与意象

**关键词**：私密、温暖、安静、亲密

**意象场景**：深夜在被窝里给朋友发私信的那种温暖安全感。窗外下着小雨，房间里只亮着一盏暖黄色的台灯，手机屏幕的光映在脸上，每一条消息都像是悄悄话。

**不是**：冷冰冰的即时通讯工具，不是企业级的协作平台。

## 视觉策略

- 整体采用新拟态柔和风格，卡片与背景同色，靠光影区分层级
- 消息气泡采用差异化设计：自己的消息用薰衣草紫渐变，对方的消息用柔和的浅色
- 头像使用纯色圆形 + 首字母，保持离线感
- 时间戳用极小的字号，不抢注意力

## 配色方案

| 角色 | 色值 | 意象来源 |
|------|------|----------|
| 主色 | #6C63FF | 薰衣草田的黄昏 |
| 主色渐变终点 | #896BFF | 暮色渐深 |
| 辅色 | #FF6584 | 珊瑚粉的温暖 |
| 背景/卡片 | #F0F0F3 | 暖灰白，像未漂白的棉纸 |
| 凹陷面 | #E8E8EB | 阴影中的纸面 |
| 自己气泡 | #6C63FF → #896BFF | 渐变紫，像夜空 |
| 对方气泡 | #FFFFFF | 干净的信纸 |
| 文字主 | #2D3436 | 墨色 |
| 文字次 | #636E72 | 铅笔色 |

## 字体排版

- 标题：28px / 800 weight
- 聊天名：16px / 700 weight
- 消息内容：15px / 400 weight
- 时间戳：11px / 400 weight
- 最后消息预览：13px / 400 weight

## 动效与交互

- 消息发送：气泡从底部滑入，轻微弹性
- 列表项点击：scale 0.97 按压反馈
- 页面转场：从右滑入

## 设计禁忌

- 不要用冷蓝色，这不是企业IM
- 不要用硬边框分割，用阴影和留白
- 不要用方角，一切都要圆润柔和
- 不要在消息气泡上加时间戳，用分组分隔
